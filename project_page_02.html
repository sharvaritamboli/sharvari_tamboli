RSI CALCULATION AND PLOTTING USING PYTHON

RSI calculation and plotting using python and yahoo_fin library

January 5, 2021

RSI:
The relative strength index (RSI) is a momentum indicator used in technical analysis that measures the magnitude of recent price changes to evaluate overbought or oversold conditions in the price of a stock or other asset.

In this I fetched historic data of the EOD prices of some symbols the data from yahoo_fin library.The data is taken from the yahoo lib and saved in a CSV like the date and hloc data. The up change and the down change are calculated between consecutive values. The mean is taken of these differences calculated. The diff is nothing but the gain or loss as if prices increase i.e. up change it’s gain if decrease i.e. down change it’s a loss. The mean calculate is put into the formula and the rsi value is cal. if RSI is over 70% it's overbought and if RSI is below 30% it's oversold.
